# Demo Vue 3

## Quick start

### Pre-requisites
```
Node.js >= 14.15
NPM >= 7.0
```

### Project setup

```bash
$ git clone git@github.com:jeferson-sb/demo-vue3.git
$ cd demo-vue3
$ npm install
```

### Usage

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Linting

```bash
npm run lint
```

## Contribution

1. Fork the repository
2. Switch to a new breach: `git checkout -b feat/awesome-feature`
3. Add files: `git add .`
4. Commit your changes: `git commit -m "feat: add XYZ"`
   1. We follow the [Conventional Commits](https://gist.github.com/joshbuchea/6f47e86d2510bce28f8e7f42ae84c716) format.
5. Push your new branch: `git push origin feature/awesome-feature`
6. Open a Pull Request and wait for approval
